# Privacidade e dados editoriais

## Principios

O produto coleta somente os dados necessarios para operar catalogo, favoritos, progresso, compras, entrega e suporte. Dados de Telegram sao usados para o fluxo autorizado e nao para inferir atributos sensiveis.

## Conteudo LGBTQIA+

A classificacao LGBTQIA+ e um atributo editorial manual da obra. O sistema nao deduz identidade, orientacao ou qualquer outro atributo sensivel a partir de comportamento, busca, nome ou linguagem do usuario.

## IA e analytics

A Express IA recebe somente o contexto minimo permitido por agente. Sugestoes sao rascunhos e exigem revisao humana. Analytics deve usar eventos minimos, sem texto de busca associado desnecessariamente a uma pessoa e sem registrar segredos, tokens ou URLs protegidas.

## Acesso e seguranca

Pagamentos, autorizacao, entrega Telegram e acesso a conteudo protegido sao validados no backend. Segredos ficam em variaveis de ambiente. Solicite exclusao ou exportacao de dados pelos canais de suporte definidos pelo produto.

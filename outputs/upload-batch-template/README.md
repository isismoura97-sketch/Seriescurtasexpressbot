# Modelo da pasta de videos

Use uma pasta local com os videos originais e escolha um destes formatos de nome:

## Formato recomendado

- `id-da-serie.mp4`

## Formato alternativo

- `titulo-da-serie.mp4`

## Extensoes aceitas

- `.mp4`
- `.m4v`
- `.mov`
- `.mkv`
- `.webm`
- `.avi`

## Como rodar o upload em lote

Use o comando de upload em lote do projeto apontando para a pasta desejada.

# Guia de upload dos videos

Use preferencialmente o nome do arquivo pelo `ID da serie`.

Formato recomendado:

```text
id-da-serie.mp4
```

Exemplo:

```text
series-id-1.mp4
```

## Se voce quiser usar o titulo no nome do arquivo

Tambem funciona, por exemplo:

```text

```
